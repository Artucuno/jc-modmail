Running Instructions
---

***Prerequesits: Node.js v15+, MongoDB***

1. Run `git clone https://github.com/wingio/jc-modmail` in a directory of your choosing.
2. Install dependencies with the package manager of your choice.
   * npm: `npm i`
   * pnpm: `pnpm i`
   * Yarn: `yarn add`
3. Start the bot!
   * npm: `npm run start`
   * pnpm: `pnpm start`
   * Yarn: `yarn run start`
